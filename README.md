# Radio Online - Reproductor Web

Una aplicación web de una sola página (Single Page Application) diseñada para radios online. Es una solución simple, moderna y efectiva para que los oyentes puedan escuchar la transmisión en vivo desde cualquier dispositivo.

## Características

- 📻 **Reproductor de Audio:** Permite escuchar la radio en vivo directamente desde la web.
- 🖼️ **Sección de Auspiciantes:** Espacio dedicado en el lateral derecho para mostrar imágenes o logos de los patrocinadores de la radio.
- 💬 **Contacto por WhatsApp:** Acceso directo para que los oyentes puedan enviar mensajes y pedir canciones en tiempo real con solo un clic.
- 📡 **Conexión de Streaming:** Integración sencilla con el servidor de streaming y el programa de radio habitual.

## Tecnologías Utilizadas

- **HTML5:** Estructura y semántica de la página.
- **CSS3:** Diseño, maquetación y sección de auspiciantes.
- **JavaScript:** Funcionalidades del reproductor y la interactividad.
